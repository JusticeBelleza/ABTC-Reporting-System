## Executive Overview

The **ABTC Reporting System** is a secure, web-based tool engineered to streamline data collection and reporting for Animal Bite Treatment Centers. The system is designed with a "privacy-first" architecture, ensuring strict adherence to data protection standards.

**Key capabilities include:**
* **Zero PII Collection:** The system does **not** collect or store Patient Personally Identifiable Information (PII).
* **Focused Metrics:** Data collection is strictly limited to aggregate animal bite case numbers and necessary personnel contact details (Name, Designation, Email, Contact Number).
* **Efficiency:** Delivers a high-performance interface for rapid record-keeping and reporting.

---

## Key Features

### Operational Capabilities
-   **Case Management:** Streamlined submission and tracking of animal bite incident reports.
-   **Data Aggregation:** Automated compilation of case statistics by reporting period.
-   **Personnel Accountability:** comprehensive logging of reporting employee details for audit trails.

### Technical Performance
-   **High-Performance Frontend:** Built on **React** and **Vite** for rapid load times and responsiveness.
-   **Scalable Backend:** Powered by a robust **Node.js** runtime environment.
-   **Secure Infrastructure:** Database management and user authentication are secured via **Supabase**.
